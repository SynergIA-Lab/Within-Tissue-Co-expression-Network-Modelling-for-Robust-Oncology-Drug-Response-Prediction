# Within-Tissue Co-expression Network Modelling for Robust Oncology Drug Response Prediction

**Ríos-Cadenas M., Segura-Carmona I., López-Fernández A., Gómez-Vela F.A.**  
SynergIA Laboratory (SIALAB), Universidad Pablo de Olavide, Seville, Spain.

*Network Modeling Analysis in Health Informatics and Bioinformatics* (under review)

---

## Overview

This repository contains all analysis code to reproduce the results of the paper. The pipeline constructs tissue-aware co-expression network features from pan-cancer cell line expression data (CCLE) and pharmacological endpoints (GDSC2), evaluates their predictive value for drug sensitivity, and validates findings in an independent cohort of drug-treated cell lines, organoids, and patient-derived xenografts (GSE255958).

The pipeline proceeds through four stages:

1. **Data acquisition & preprocessing** — gene filtering, ComBat batch correction, tissue residualisation
2. **Tissue-aware network feature extraction** — WGCNA modules, edge disruption profiles, topological metrics
3. **Predictive modelling & statistical evaluation** — cross-validated performance, Wilcoxon tests, permutation testing
4. **Interpretability & validation** — SHAP analysis, YAP1/TEAD1 centrality audit, external validation

## Data

All datasets are publicly available. Download them and place them in `data/raw/` before running any script.

| Dataset | Source | Description |
|---------|--------|-------------|
| CCLE 20Q1 gene expression | [DepMap portal](https://depmap.org/portal/) | `CCLE_expression.csv` — log₂(TPM+1), 1,249 cell lines × 19,144 genes |
| GDSC2 drug sensitivity | [Wellcome Sanger](https://www.cancerrxgene.org/) | `GDSC2_fitted_dose_response.xlsx` — 243,466 dose-response measurements |
| Cell Model Passports | [Sanger CMP](https://cellmodelpassports.sanger.ac.uk/) | `model_list_20230110.csv` — CCLE–GDSC2 identifier mapping |
| GSE255958 RNA-seq | [NCBI GEO](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE255958) | Cell lines, organoids, PDX from Haderk et al. (2024) |

## Execution order

Run scripts sequentially in the order listed. Each script reads outputs from previous steps.

### Stage 1 — Preprocessing

```bash
python scripts/01_preprocessing.py
```

**Inputs:** `data/raw/` (CCLE, GDSC2, Cell Model Passports)  
**Outputs:** `data/processed/`
- `X_expr_matched.csv` — batch-corrected expression matrix (660 lines × 14,658 genes)
- `X_expr_residualized.csv` — tissue-residualised expression matrix
- `y_matched.csv` — drug sensitivity endpoints (AUC / LN IC₅₀) for 3 drugs
- `cell_line_metadata.csv` — tissue labels and identifiers
- `gse255958_expression.csv` / `gse255958_metadata.csv` — external validation set

---

### Stage 2 — Network feature extraction

**2a. WGCNA modules** (R)

```bash
Rscript scripts/02a_wgcna_v3.R
```

**Inputs:** `data/processed/X_expr_matched.csv`, `cell_line_metadata.csv`  
**Outputs:** `networks_v3/wgcna_v3/`, `features_v3/module_activity_v3.csv` (660 × 510)  
**Runtime:** ~30–60 min depending on hardware

**2b. Edge disruption profiles**

```bash
python scripts/02b_edge_disruption_v3.py
```

**Inputs:** `data/processed/X_expr_matched.csv`, `cell_line_metadata.csv`  
**Outputs:** `networks_v3/within_tissue/`, `features_v3/edge_disruption_within_tissue.csv` (660 × 65)

**2c. Topological metrics**

```bash
python scripts/02c_topology_v3.py       # raw features (660 × 72)
python scripts/02c_topology_zscore.py   # within-tissue z-scored features (660 × 72)
```

**Inputs:** `data/processed/X_expr_matched.csv`, `features_v3/topo_features_v3.csv` (zscore reads from v3)  
**Outputs:** `features_v3/topo_features_v3.csv`, `zscore/features/topo_features_zscore.csv`

---

### Stage 3 — Predictive modelling

```bash
python scripts/03_modeling_v3.py       # generates baseline and network scores (needed for Stage 3 zscore)
python scripts/03_modeling_zscore.py   # PAPER RESULTS: z-scored feature sets + Wilcoxon + tissue QC
```

**Inputs:** `data/processed/`, `features_v3/`, `zscore/features/`  
**Outputs:**
- `models_v3/cv_results/cv_scores_v3.csv` — fold-level Spearman ρ for all v3 feature sets
- `zscore/models/cv_results/cv_scores_zscore.csv` — fold-level Spearman ρ (paper Table 3)
- `zscore/models/cv_results/wilcoxon_zscore.csv` — Wilcoxon + FDR results (27 tests)
- `zscore/models/cv_results/tissue_accuracy_zscore.csv` — tissue QC for all feature sets

> **Important:** `03_modeling_zscore.py` reuses baseline and Modules/EdgeDisrupt scores from `cv_scores_v3.csv`. Run `03_modeling_v3.py` first.

---

### Stage 4 — Interpretability and validation

**4a. SHAP + tissue-stratified analysis**

```bash
python scripts/04_shap_v3.py
```

**Inputs:** `data/processed/`, `features_v3/`  
**Outputs:** `models_v3/shap/`, `models_v3/cv_results/tissue_stratified_v3.csv`

**4b. External validation (GSE255958)**

```bash
python scripts/05_validation_v3.py
```

**Inputs:** `data/processed/gse255958_*.csv`, `data/raw/GSE255958/`, `data/processed/X_expr_matched.csv`  
**Outputs:** `models_v3/cv_results/gse255958_roc_v3.csv`, `figures_v3/05_roc_v3.png`

**4c. Robustness checks**

```bash
python scripts/06_robustness_v3.py
```

**Inputs:** all processed features and model scores  
**Outputs:** tissue accuracy QC, permutation test results, threshold sensitivity analysis

---

### Supplementary experiments

```bash
python scripts/supplementary_experiments.py
```

Runs six experiments in sequence:
1. SHAP on unconfounded features (Combined_zscore + Topology_zscore) for osimertinib
2. YAP1/TEAD1 SHAP distribution by drug sensitivity quartile (all 3 drugs)
3. Ablation study: Combined_noEdge = Modules + Topology_zscore (no EdgeDisrupt)
4. Sensitivity analysis: exclude kNN-imputed cell lines
5. WGCNA module stability (note: implemented in a separate R script; see `results/supplementary/wgcna_stability_analysis.R`)
6. PCA baseline vs Modules comparison (K=226 components, 80% variance)

**Inputs:** all feature matrices and model scores  
**Outputs:** `results/supplementary/` — tables and figures for Online Resources 1–3

## Reproducibility notes

- All Python scripts use `random_state=42`. Results are fully deterministic across runs with the same scikit-learn version.
- The R script uses `set.seed` implicitly through WGCNA's internal seeding. Minor module-level variation is possible across R versions but module preservation was confirmed robust (100% preservation rate in bootstrap analysis).
- `03_modeling_zscore.py` concatenates new z-score CV scores with scores read from `cv_scores_v3.csv` (generated by `03_modeling_v3.py`). Both files must exist before running Wilcoxon tests.
- Tissue accuracy values cited in the paper derive from `LogisticRegression(C=0.1, solver='saga')` throughout. Using a different classifier (e.g. RidgeClassifier) will produce slightly different values.
- The `supplementary_experiments.py` script runs all six experiments in sequence. Experiment 6 (PCA baseline) depends on `cv_scores_v3.csv` being present.
